# React Music Search

A React single-page app for searching albums and previewing tracks with the iTunes Search API.

## Highlights

- Searches albums through the iTunes Search API using `fetch`.
- Uses React Router to switch between album results and track-detail pages.
- Displays loading and error states for API requests.
- Lets users play and stop short track previews.
- Uses Bootstrap and FontAwesome for UI styling.

## Tech Stack

- React
- JavaScript
- React Router
- Bootstrap / React Bootstrap
- FontAwesome
- iTunes Search API

## Run Locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Project Context

This project was built as a client-side development exercise and cleaned for portfolio presentation. The public version keeps the app source code and removes course-specific prompts, tests, and grading materials.
