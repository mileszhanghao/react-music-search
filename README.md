# React Music Search Product Prototype

An API-driven React single-page app for searching albums and previewing tracks with the iTunes Search API.

## Product Capabilities

- Searches albums through the iTunes Search API using `fetch`.
- Uses React Router to switch between album results and track-detail pages.
- Displays loading and error states for API requests.
- Lets users play and stop short track previews.
- Uses Bootstrap and FontAwesome for UI styling.
- Demonstrates a consumer-search workflow with API-backed results and track-level preview interactions.

## Tech Stack

- React
- JavaScript
- React Router
- Bootstrap / React Bootstrap
- FontAwesome
- iTunes Search API

## Run Locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Project Context

This project is presented as a frontend product prototype. The public version keeps the app source code and removes non-portfolio scaffolding.
